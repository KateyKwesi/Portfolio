## Commands


No test suite exists.

## Architecture

Single-page React 19 + TypeScript app bundled with Vite 7. No router — sections render sequentially in `App.tsx`:

```
Navbar → Hero → About → Projects → Experience → Testimonial → Contact → Footer
```

- `src/layout/` — Navbar, Footer
- `src/sections/` — one file per page section
- `src/conponents/` — shared UI components (folder name is intentionally misspelled; don't rename without updating all imports)
- `src/assets/projects/` — project screenshot images

## Key Details

- **Tailwind v4** — no `tailwind.config.js`; configured via the `@tailwindcss/vite` plugin and `index.css` directly.
- **EmailJS** — contact form sends email client-side via `@emailjs/browser`. Keys are in `.env.local` (not committed).
- No state management library; all state is local component state.
